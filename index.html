The World Wide Web

The world wide web is the world’s largest network. It consists of about 30 billion webpages, which are all based on HTML (= hypertext markup language). These pages are stored on servers all over the world, which connect to the world wide web through HTTP. The world wide web can be accessed by end users through internet browsers on their PC.

HTML

HTML is based on several key elements:
  • Text content = what you see
  • Markup = what it looks like
    • The markup is based on tags, which always come in pairs, an opening tag and a closing tag (except for void tags, which come alone)
    => They always use the same structure: < name > content< /name >
  • References to other documents (e.g. images, videos, etc ...)
  • Links to other pages
